# JokeGenerator - Joke Generator Web Application

A simple and fun web application that generates random jokes to brighten your day! Built with vanilla JavaScript and powered by the JokeAPI.

## Features

-   Random joke generation from multiple categories.
-   Clean and responsive user interface.
-   Instant joke refresh with a single click.
-   No dependencies or frameworks required.
-   Fast and lightweight.

## Installation

Clone the repository to your local machine:

```bash
git clone https://github.com/Balsha98/Repository-JokeGenerator.git
```

Navigate to the project directory:

```bash
cd Repository-JokeGenerator/joke-generator
```

Open the project in your browser:

```bash
# Simply open index.html in your preferred browser,
# or use a local server like Live Server in VS Code.
```

## Usage

1. Open the application in your web browser.
2. A random joke will be displayed automatically.
3. Click the **Refresh** button to generate a new joke.
4. Enjoy unlimited laughs!

## API Reference

This project uses the free [JokeAPI v2](https://jokeapi.dev/):

-   **Endpoint**: `https://v2.jokeapi.dev/joke/any`
-   **Method**: GET
-   **Response**: A random jokes from various categories.
